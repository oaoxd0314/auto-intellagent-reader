# 自卑感

自卑感 ( `Minderwertigkeitsgefuhl`) 在德文中，是由「價值 ( wert )」「較少 ( minder )」「感覺 (Gefuhl )」組成的，也就是說自卑感是等同於對**自我的價值判斷較低**



折磨自己的自卑感是**主觀的解釋，並不是客觀的事實**，就像身高是個可以客觀測量的數值，很多人會因為身高而自卑，但這還是取決於你用甚麼角度看待 ( 主觀的解釋是甚麼 )。



雖然自己賦予的意義就是一廂情願的說法，但**主觀的好處就是可以任由自己選擇**，只要能夠找到新的衡量標準，例如類似「身高不高會更有親和力，會更容易打好和他人的關係」，就可以減少自己的心理內耗。



> **所謂的價值，必須在社會的脈絡中才得以體現，如果沒有他人的話，就不會因為自卑感而困擾**
