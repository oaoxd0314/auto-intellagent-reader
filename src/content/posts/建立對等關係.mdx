# 建立對等關係

當我們收到稱讚或是責罵，不過就是【要給糖，還是抽鞭子】的差異，背後的目的都是操縱。



希望得到某人的稱讚，或是稱讚他人，都是在人際關係中採取縱向關係的證據。

- 我希望收到他人的稱讚 → 在意他人的想法，但他人的想法應該是[他人的課題](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/0af715be-a2c9-4deb-9de3-16f32ad50f48)

- 當我們稱讚他人，是以縱向關係看待人際關係，認為對方比自己低一等，所以就介入 ( 評論 ) 他人的課題。

   - 總的來說就是「乾你屁事」

      ![image.png](./建立對等關係-assets/image.png)



阿德勒心理學否定縱向關係，提倡人際關係應該要是橫向關係，不是上對下，而是對等的關係。縱向關係會導致課題的介入，會導致人際關係的複雜，就沒辦法獲得自由。



建立橫向關係重要得是，在意識上是對等的，而且該堅持的地方就堅持，坦坦蕩蕩不退縮。

## 橫向關係的鼓勵

要建立橫向關係就不能用稱讚，而是採取鼓勵。



因為獲得稱讚會產生依賴，會變得像是【做了這件事是因為想獲得他人的稱讚】，[當獲得稱讚成為目的，最終就無法獲得自由，最終都是在迎合別人的價值觀](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/d9ee934f-087e-45c0-8d91-925fd1e7d77e)。



鼓勵和稱讚不同的地方在於，鼓勵不評價他人，如果想構築好的橫向關係，應該要的是更坦承的感謝「多虧有你幫忙」「謝謝」等，用來表達更誠摯的感謝、敬意或喜悅。



當他人聽到感謝時，就會明白自己對他人是有價值的，從而獲得歸屬感。
