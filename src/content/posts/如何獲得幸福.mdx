# 如何獲得幸福

1. 所有的煩惱都來自於[「人際關係」](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/1d7cc349-89d6-44bf-8e66-9a8debfa9b1e)

2. 透過[課題分離.md](./課題分離.md)，理解甚麼是可以改變的，甚麼是不必在意的，讓自己的**主觀視角變的單純**

3. 認知自己的[世界觀](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/14ada498-0e50-4839-902a-70a30d46e803)（自己關心的區域）

4. [貫徹與認知自我](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/d9ee934f-087e-45c0-8d91-925fd1e7d77e)

5. [建立對等關係.md](./建立對等關係.md)，採取[正確的態度](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/b9a03a7b-2604-452e-9849-c541cc36cd1b) ( 我可以給這個人甚麼 ) 與他人相處

6. [把他人當作同伴，在自己的世界觀獲得歸屬感 ](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/f030b8a2-46cc-4f79-b2bd-9ec9512392a7)

7. [活在當下](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/3187d9b5-6bf0-463e-9fb0-135ec91ebb05)不斷地為自己的熱愛貢獻
