# 個體心理學 

個體的意思是，不可被分割的最小單位

- 將心理與生理，意識無意識都看作一整個整體

   - 人的行為、思想、感受是不可拆分的

- 人是一個統合的整體，並不受單一情感而驅動

- 忠實於自己的價值觀，所以當人表現出憤怒失落開心時，是選擇用「整體的我」表現的

- 如果將情感與表現切割，「是情感讓我這麼做的 / 我也不由自主」 就會陷入[人生的謊言.md](./人生的謊言.md)



個體心理學是一種探討如何獲得幸福的方法:

- 打造良好的人際關係

- 人際關係的終極目標在「社會意識」

- 社會意識是在自己所認知的「[世界](https://app.heptabase.com/aa5e5fd4-4215-4de1-90fe-de29e11f0187/card/6fb32b54-a6b0-481d-854a-f7903a34125e)」具有歸屬感



所以我們需要由個體走向群體，從而找到歸屬感

- 歸屬感是與他人建立健康的關係

- 要獲得歸屬感的方式是「貢獻」

- 要讓自己覺得，自己對於這個「群體 / 社會意識 」 是有幫助的
